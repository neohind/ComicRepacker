이 프로젝트는 2장짜리 이미지를 1장짜리로 자동 분리해주는 프로그램입니다.
보통 책 스캔의 경우 파일 갯수를 줄이기 위해서 이렇게 하는 경우가 많은데,
E-Book 뷰어로 보려고 할 때, 제대로 볼 수 없는 문제가 있어서....

아직 개발 중인 프로그램이라 실제적인 활용은 어렵지만, 당장은 사용가능합니다.

1. 사용 방법
먼저 스캔본 파일을 Load File 합니다. 여러 개로 구성되어 있다면 여러 개의 파일을 읽어옵니다.
밑의 List 박스에 전부 나열 되었으면, 상단의 Result Filename에 적당한 대표 이름을 적습니다.
Save File을 선택하면, Result Filename으로 Zip 파일을 만들게 됩니다.
해당 Zip 파일을 E-Book 뷰어에 넣으면 됩니다.

현재 가지고 있는 E-Book 뷰어는 페이지 밖에 없어서, 페이지에서만 정상적으로 보입니다.
